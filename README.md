## **What is vIDE lite?**
vIDE lite is a lightweight html/css based web editor built to be minimal, basic, and run on almost anything with an browser that can display html.
## **Why is it "lite" (lightweight)**
In simple terms vIDE is lightweight (lite) due to the simplistic and basic design of vIDE lite. Its centered around being a lightweight text-editor like IDE that can run on almost any device via. exsisting compatable browser. Which means it is stripped of heavy or unnessesary things to be able to run so lightly.
## **How do I install/get it?**
It's as easy as five simple steps!
1. Go to https://github.com/10NM01/vIDE-lite (Main page for repo).
2. Click on the Releases tab.
3. Download your html file based on your preffered theme.
4. Open the html file with a browser.
5. Enjoy!
